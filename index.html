AdminAssist - Mi Bot de Discord

Descripción

¡Hola! Soy un bot de Discord diseñado para ayudarte a gestionar tu servidor, proporcionar diversión y mejorar la experiencia de los usuarios. Mis características incluyen comandos de moderación, juegos, y más.

Información del Bot

Fecha de Creación: 19 de agosto de 2024

Creador: Aleks

Lenguajes de Programación: Python

Origen de la Idea: La idea surgió mientras estaba modificando mi servidor de Discord.

Funcionamiento: El bot está diseñado para interactuar con los usuarios en Discord, ejecutando comandos y proporcionando respuestas automatizadas a diversas solicitudes. Utiliza programación en Python para gestionar las interacciones y realizar tareas específicas de forma eficiente.

Comandos del Bot

Aquí tienes una lista de todos los comandos disponibles:

!logo - Muestra el logo del servidor.
!server_stats - Muestra estadísticas del servidor.
!versiones - Muestra las versiones del bot.
!soporte - Envía una solicitud de soporte al equipo.
!supports - Envía una solicitud de soporte al equipo.
!actualizaciones - Para ver las modificaciones del bot.
!sugerir - Para dar una sugerencia.
!warn - Para advertir a un usuario.
!ban - Para bloquear a un usuario del servidor.
!ver_sugerencias - Para ver todas las sugerencias.
!kick - Para expulsar a un usuario del servidor.
!trivia - Para que el bot te haga preguntas, esta en desarrollo esta opción.
!logo_bot - Para ver el logo del bot.
!ticket - Para crear un botón para abrir ticket.
Y otros que se encuentran en desarrollo.
